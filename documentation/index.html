About Catalizer template

Catalizer is premium onepage HTML5 template suitable for companies or individuals that want to present their work at the highest level. It is based on Twitter Bootstrap 3 framework. This template uses the latest HTML5 and CSS3 technologies.

Columns

Template is divided into rows and columns, with respective classes .row and .col-md* .col-sm*, col-lg*, col-xs*, where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12). For more information about twitter bootstrap 3.0 grid system and its columns visit this site: http://getbootstrap.com/css/#grid

Back to Top

Container

There's one more special class which is being used across all HTML files, which is .container. This class determines the maximum width of a page and also makes sure that page is center-aligned. It also determines rendering of a template on mobile devices and tablets. This class can have also different backgrounds.
